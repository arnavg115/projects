Facial Recognition

Using a convolutional neural network this program is able to recognize faces after being provided a lot of data. This uses code from many sources and a little of my own code. It uses Tensorflow and opencv, to recognize the faces and do a lot of image processing respectively.

Stringsim

A python package with many string similarity algorithms from levenshtein distance to cosine similarity. It can be used for nlp and also a sort of autocomplete system (See Ptsearcher)

Ptsearcher

A tool designed to make searching the periodic table easy and very fast. It uses a fuzzy matching system (fuzzywuzzy) to come up with the best result no matter the formatting.
